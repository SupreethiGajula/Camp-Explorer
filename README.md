# YelpCamp Project

## Overview
YelpCamp is a full-stack web application where users can create, view,
review, and manage campgrounds.

## Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
- Bootstrap
- javascript

## Features
- User authentication and authorization
- CRUD operations for campgrounds
- Reviews and ratings
- Server-side validation
- Flash messages and error handling

## Project Structure
- Routes are separated by feature (campgrounds, reviews, auth)
- Controllers handle business logic
- Models define database schemas
- Middleware handles authentication and validation
